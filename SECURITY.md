# Security Policy

The canonical OpenGantry security policy is published at **[`docs/SECURITY.md`](docs/SECURITY.md)** (also on [opengantry.ai](https://opengantry.ai)).

**Report vulnerabilities:** use [GitHub Private Vulnerability Reporting](https://github.com/jeger-ai/opengantry/security/advisories/new) — do not open public issues for undisclosed security problems.
